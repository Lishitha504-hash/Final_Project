# 🌱 AgroBot

AgroBot is an AI-based farming assistant that helps farmers identify **crop diseases** and **nutrient deficiencies** using plant leaf images. It also provides simple farming guidance in a farmer-friendly way.

---

## ✅ Key Features

- 📷 **Image-Based Crop Diagnosis**
  - Upload or capture a leaf image
  - AgroBot detects:
    - Disease symptoms
    - Nutrient deficiency signs

- 🌾 **Farming Assistance**
  - Gives simple guidance for plant care
  - Helps farmers take early action to protect crops

- 🧠 **Works with Both Local Model + API**
  - Local AI model can work offline
  - If the local model is unsure, online API analysis can be used for higher accuracy

- 🌍 **Multi-Language Support**
  - Can respond in multiple Indian languages for easy understanding

---

## 🧪 How It Works

1. User uploads a plant/leaf image  
2. AI analyzes the image  
3. AgroBot identifies:
   - Healthy or infected leaf
   - Disease name or nutrient issue  
4. AgroBot provides simple suggestions to fix the problem
